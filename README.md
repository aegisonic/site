# Aegisonic
Static site for Aegisonic, a project seeking to provide an equitable acoustic experience for those living with heightened sound sensitivity and autism.

## Content

### Blog posts

Located within `content/blog`. To add a post, create a new text (.md) file, with the following frontmatter to the top of the document. Placeholders should be populated with the relevant information for that post. 

```
---
title: "{{title}}"
date: {{date}}
description: "{{description}}"
draft: true
---
```

### Photo gallery

Located within `content/gallery`. 

The name of each image file will be treated as the image caption within the site.
